PWA Icon Requirements
====================

To complete your PWA setup, you need to create the following icon files:

Required sizes:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

How to generate icons:
1. Create a high-resolution logo (at least 512x512px)
2. Use a tool like https://www.pwabuilder.com/imageGenerator
3. Or use ImageMagick: 
   convert logo.png -resize 72x72 icon-72x72.png
   convert logo.png -resize 96x96 icon-96x96.png
   ... (repeat for all sizes)

Place all generated icons in the /public directory.

Note: The app will still work without these icons, but they're needed for:
- Home screen installation
- App switcher appearance
- Splash screens
